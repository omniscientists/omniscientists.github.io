What it does

Not a chatbot with a code box. Working scientists.

Omniscientist keeps state, sees its results, and reasons over real computation — the way someone actually works at a live notebook.

Persistent Python kernel

Variables, imports and data survive across cells, so the agent builds on its own work instead of starting cold every prompt.

Scientific rendering

See the result, not a paragraph about it. Proteins in 3D, molecules, genome tracks, figures and notebooks render as first-class artifacts.

Agent orchestration

A coordinator fans hard problems out to domain specialists — and routes their work back through a reviewer before it reaches you.

Provenance & reproducibility

Each figure and file links back to the exact cell that produced it — inspectable, re-runnable, honest by construction.

Skills & MCP tools

A library of composable skills plus MCP connectors let it pick up new methods, datasets and instruments on demand.

Local & open

Runs as a daemon on your own hardware, talks to any model, and the whole stack is Apache-2.0. No lock-in, no data exhaust.

Under the hood

One local daemon. A full research stack.

A thin web workspace over a Rust daemon that runs the agent loop, a live Python kernel, a skill & tool layer, and an artifact store that remembers where everything came from.
