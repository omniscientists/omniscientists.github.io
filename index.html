AI scientists that run the experiment.

Omniscientists is an open-source, local-first research agent — a coordinator and its domain specialists. They write and run real code in a persistent kernel, render the result, and trace every figure back to the cell that produced it.

What the agents actually do

Not a chatbot with a code box. A working group that keeps state, sees its results, and reasons over real computation.

a

Persistent kernel

Variables, imports and data carry across cells — it builds on its own work.

b

Scientific rendering

See the structure, not a paragraph about it — proteins, molecules, genomes.

c

Agent orchestration

A coordinator delegates to specialists and routes their work through a reviewer.

d

Provenance

Every figure and file links back to the exact cell that produced it.

e

Skills & MCP

A library of composable skills and MCP tools — new methods on demand.

f

Local & open

Runs as a daemon on your hardware. Your data stays yours; the stack is open.
