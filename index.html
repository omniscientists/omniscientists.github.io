AI scientists that run the experiment.

Omniscientists is an open-source research agent — a coordinator and its domain specialists. They write and run real code in a persistent kernel, render the result, and trace every figure back to the cell that produced it.

// 02 · Capabilities

The anatomy of a computational scientist

MODULES 06
STATUS ▸ active
01 / 06

Persistent kernel

State persists across cells — variables, imports, intermediate data — so reasoning compounds rather than resetting each turn.

02 / 06

Scientific rendering

Results materialize as structures, not prose — proteins, molecules and genomes, rendered in place.

03 / 06

Agent orchestration

A coordinator decomposes the problem across domain specialists, then reconciles their results through a reviewer.

04 / 06

Provenance

Every artifact resolves to the exact cell that produced it — inspectable, re-runnable, auditable.

05 / 06

Skills & MCP

Composable skills and MCP connectors extend its methods, datasets and instruments on demand.

06 / 06

Local & open

Runs as a daemon on your own hardware — your data never leaves, and the stack is open end to end.

One command to a live workspace.

The daemon opens a local workspace you drive directly — every hypothesis executed, not just described. Bring the model you trust; keep the data you own.

omniscientists — session▸ localhost:4100
$ git clone https://github.com/omniscientists/omniscientists
$ omniscientists          # start daemon + open workspace
→ workspace ready at http://localhost:4100
